A simple text transformation tool for switching between sentence, capital, lower and upper case built with JavaScript

## Installation

Clone this repo and npm install

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:8080`.

### Production build

```bash
npm run build
```
